# Tanks-Unity  [![forthebadge](https://forthebadge.com/images/badges/built-with-love.svg)](https://forthebadge.com)

This project is a 3D tank battle game developed using Unity, inspired by the official Tanks: Make a Battle Game for Web and Mobile course from Unity Learn The game demonstrates core game development concepts such as AI navigation, player controls, physics-based shooting, and enemy behavior. Enemy tanks use NavMesh pathfinding to intelligently navigate the environment and engage the player. The project serves as a hands-on exploration of Unity game mechanics, AI systems, and scene management, making it a useful learning resource for understanding how real-time game logic and autonomous agents are implemented.

## Key Features

- Player-controlled tank with movement and shooting mechanics
- Enemy tanks with AI navigation using Unity NavMesh
- Projectile physics and explosion effects
- Health and damage system
- Basic game UI and scene management
- Modular scripts for scalable gameplay mechanics
